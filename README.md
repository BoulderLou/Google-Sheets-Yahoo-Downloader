# Google-Sheets-Yahoo-Downloader
Google Sheets Yahoo JSON Stock downloader.

syntax:

=YHISTORICAL("symbol", "start date","end date")

example:

=YHISTORICAL("NVDA", "1/1/2023","12/31/2024")
